Domain names and ICANN

 When entering a domain name into the web browser it send a request to a global network callled the DNS. The DNS server look up the servers associated with the domain name and forward the request. Examples of domain names include the following: .com, .org, net, .tv, .info, .io, and more. In other words, domain is the address that individuals type into a browser address bar to find a website.

Definitions

  • Domain Name - Domain name is the address of your website that people type in the browser URL bar to visit your website. It is typically what the user searches for in the web browser.
