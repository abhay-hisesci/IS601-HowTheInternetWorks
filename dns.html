DNS (Domain Name System)

 DNS associates names like WWW with the corresponding addresses. The computer uses the DNS find the address to a specific website. Designing a system for billions of devices to find any one of billions of websites requires having more than one DNS. DNS server are connected in a distributed hierarchy and are divided into zones. Because DNS is open, it is susceptible to attacks like DNS Spoofing.

Definitions

  • DNS - Also known as Domain Name System is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network. It associates various information with domain names assigned to each of the participating entities.
