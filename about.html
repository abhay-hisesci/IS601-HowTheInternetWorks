About the Project

This project is part of the coursework for IS601 at NJIT that requires team members to create a functional website that covers topics about how the internet works. The topics covered will enlighten website visitors on the inner workings of the internet and how it functions.

Project Goals

Project goals are to utilize Bootstrap and create a minimum of 10 commits along with 1 image per page. Each page should cover a maximum of one topic and should include a home page and about page for a project description.
